判决

宁缺承认 Typora 1.13.6。

不是因为它功能最多。正相反：这一次，它最少要求作者先处理软件。

Typora 打开文稿后，屏幕中央只有文字。没有教程工作区，没有默认文件树，没有为了证明专业而同时展开的面板。标题、正文、引用、列表和行内代码各自站稳；Markdown 标记只在编辑所需的位置出现。

这不证明 Typora 是“最佳 Markdown 编辑器”，也不证明所有作者写得更快。它只回答本期那一个时刻：一篇本地中文稿已经在手里，工具能不能退到文字之后。

判据与范围

本期只比较 PC／Linux x86_64 默认配置下的 Typora 1.13.6、MarkText 0.19.1 与 Zettlr 4.6.0。任务是一篇单独的本地中文 Markdown 短稿，不是功能排行榜。

三款都从独立 HOME、配置、用户数据和同一份 717 字节样本开始；不安装插件、主题、词典或同步。固定流程包括完整阅读、补写、改引用与列表、跨结构移动光标、主动保存、退出和全新进程重开。

宁缺依次看五件事：文字是否成为屏幕上最重要的东西；结构编辑是否连续；工具是否主动退场；保存和重开是否可信；试用、许可与长期代价是否诚实。

三次流程分别持续 1223、1224、1239 秒，三款最终副本都以目标 SHA-256 由全新进程重开。Typora 本次计时自动化的源码注入没有在同一次进程落盘；失败已显著披露。此前独立 GUI 精确保存、本次完整时长和目标字节重开共同构成本期证据。

承认 · ADMIT

Typora 1.13.6

它没有先把作者变成管理员

全新配置直接进入单文稿。默认窗口只有传统菜单、正文和底部状态栏；没有 onboarding、教程、工作区或额外分栏。正文宽度、留白和层级让中文句子处在屏幕中央，而不是让应用先展示自己能做什么。

它让结构出现，但不常驻

补写、引用和列表留在同一张所见即所得表面。非当前块的 Markdown 标记退场；引用是一条克制的竖线，列表保留清楚圆点。作者仍知道自己在编辑什么，却不必一直面对语法的脚手架。

代价必须写在承认旁边

Typora 是商业软件。本期使用官方 15 天试用，没有购买、注册、输入邮箱或许可证；界面底部明确显示“未激活”。这不是免费推荐。试用结束后的购买、账号和激活，是承认必须携带的长期代价。

否决 · REJECT

MarkText 0.19.1

它没有输在可靠性，它输在最后一点没有退完。

正式隔离运行同样是一张干净的单文稿画布。它免费、开源，目标字节由应用精确保存并由全新进程重开。顶部路径、字数、汉堡菜单和当前段落符号比 Typora 更常提醒主审“正在使用编辑器”；精确控制结构时，还要意识到视觉面与源码面之间的切换。

这点差异很小，也未必值得所有人付费。若不愿接受商业许可、账号或激活，MarkText 是本期不付费时的直接答案。

否决 · REJECT

Zettlr 4.6.0

它不是差的写作器；它只是先把作者请进了一间工作室。

全新配置先出现 onboarding。点击“跳过”后，教程工作区、文件树、多个标签、右侧分栏、工具栏和状态信息同时进入屏幕。标题旁的 # 与其他结构标记更持续可见。

对研究、引用、出版和多文档组织，这些不是杂音，而是控制力。但本期只有一篇短稿。主审不需要工作台，只需要一张能继续写下去的纸。这项第三名不外推到论文、知识管理或出版流程。

事实账

三款候选在本期固定流程中的结果
候选时长新进程重开最强优点主要代价
Typora 1.13.61223 秒通过单一表面最克制商业许可、试用、激活
MarkText 0.19.11224 秒通过免费开源、足够安静模式意识与状态信息
Zettlr 4.6.01239 秒通过研究与出版组织能力教程、工作区与持续标记

三款都能完成本期目标。这个表不构成功能、效率或稳定性排行榜。

最强反例

MarkText 已经足够接近，而且免费、开源、保存可靠。Typora 的优势主要是同一表面里的视觉克制，不是压倒性的功能或效率胜利。

宁缺仍承认 Typora，因为本期判的不是“免费条件下最划算”，而是哪一款在单篇中文稿前最愿意后退。这个优势很小，但每次面对文稿时都会出现；是否值得付费，必须由读者自己决定。

边界、利益与复审

适合：只想打开一篇本地 Markdown、连续写完再关掉窗口，重视正文秩序与标记退场，并愿意为细小但持续的克制付费的人。

不适合：不接受商业许可、账号或激活，依赖可见 Markdown 标记，或需要多文档、引文、研究资料与出版工作台的人。不付费时选 MarkText；需要工作台时，Zettlr 更可能合适。

利益披露：Typora 使用官方试用；MarkText 与 Zettlr 使用公开发行制品。宁缺、Codex 与公开主审同三款候选及相关权利人不存在购买、赠送、广告、佣金、联名、投资或其他商业关系；候选方均未参与、批准或背书本判词。

未覆盖什么

本期没有比较 PDF 导出、引用管理、学术工作流、云同步、协作、插件生态、AI 写作、移动端、Windows、macOS、数万字滚动、章节导航、搜索替换、崩溃恢复或长期疲劳。267 字固定稿不能证明长篇小说、论文或数万字项目中的优胜。
